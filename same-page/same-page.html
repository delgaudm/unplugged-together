How to play

  1. Everyone gets paper and a pencil. Choose a scoring mode, tap Deal a prompt, and set the phone in the middle.
  2. Each player secretly writes one answer that completes the prompt. No conferring, hinting, or campaigning.
  3. When time runs out, reveal answers together. The table decides which answers are close enough to count as the same idea.
  4. Score the round using the mode shown on the screen, then deal the next prompt.
  5. Highest score after the final round wins. Ties are settled by one sudden-death prompt.

For close calls, be generous with wording and strict with ideas. "A ghost" and "ghost" match; "monster" probably does not.

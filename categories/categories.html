How to play

  1. Everyone gets paper and a pencil. Number your page to match the list, tap Deal a round, and set the phone in the middle.
  2. The phone shows one letter and a list of categories. For each category, write one answer that starts with that letter before time runs out.
  3. When the screen dims, pencils down. Go category by category, reading answers aloud.
  4. Any answer written by two or more players is crossed out by everyone. Each surviving answer scores 1 point. The table votes on whether a creative answer counts — majority rules, laughter is admissible evidence.
  5. Highest total after an agreed number of rounds wins. A new round deals a fresh letter and fresh categories.

House rule for the brave: an answer whose words all start with the letter scores 1 per word ("Barking Basset" under B = 2).
